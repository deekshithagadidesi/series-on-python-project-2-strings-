## 📌 Overview

A beginner-friendly Python project that demonstrates various **String Operations** and **User Input Handling**.

This project analyzes a student's profile and performs different text-processing tasks such as:

✨ Converting text to uppercase and lowercase  
✨ Finding string length  
✨ Accessing characters using indexing  
✨ Reversing strings using slicing  
✨ Generating formatted introductions using f-strings  
✨ Counting character occurrences  
✨ Checking character presence  
✨ Removing extra spaces  
✨ Splitting text into words  
✨ Replacing words in a string  
✨ Counting vowels
